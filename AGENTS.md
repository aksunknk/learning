# AGENTS.md

## Cursor Cloud specific instructions

### 概要
静的な学習サイト（"Code Foundations"）です。ビルド・パッケージマネージャ・バックエンドは存在せず、構成ファイルは3つのみです。

- `index.html` — 全ページ本体（各章・レッスン・クイズ・パズルの内容を含む）
- `app.js` — クイズ／タブ／パズル等のインタラクションロジック（バニラJS、外部依存なし）
- `style.css` — スタイル

### 依存関係
- ローカル依存・`node_modules`・ロックファイルは無し。インストール手順は不要です。
- 唯一の外部依存は Google Fonts（CDN）で、ネットワーク不通でもフォントがフォールバックするだけで機能に影響しません。

### 開発サーバーの起動（run）
`file://` ではなく HTTP 経由で開く必要があります（相対パスの `app.js`/`style.css` 読み込みのため）。リポジトリ直下で次を実行します。

```
python3 -m http.server 8000
```

その後ブラウザで `http://localhost:8000/` を開きます。ホットリロード機構は無いため、ファイル変更後はブラウザを手動リロードしてください。

### Lint / Test / Build
- 専用の lint・テスト・ビルドの仕組みは存在しません（該当スクリプト無し）。
- 検証は上記サーバーを起動し、ブラウザでクイズ回答等の操作を行う手動確認で実施します。
